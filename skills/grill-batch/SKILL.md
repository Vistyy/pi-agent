# Grill batch

Use this only when a `grill-me` or `grilling` session is already in progress.

Stop asking questions one by one.

Gather the remaining grill questions into a list.

For each question, give the recommended answer.

Do not ask a new question.

The batch is complete only when every remaining question has a recommended answer.
